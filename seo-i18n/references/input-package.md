# Input Package

Provide one directory that includes all available materials, then share the directory path.

## Required

- Scenario source: design artifact(s) or code file(s)
- Keyword input: keyword list/table for SEO targeting
- Prohibited-term input: banned words or phrases list

## Optional but Recommended

- Competitor copy references
- Existing page copy to be rewritten
- Manual requirement notes in `.md`
- Route map and output naming rules

## Directory Convention

Use a structure similar to:

```text
input/
  design-or-code/
  keywords/
  prohibited/
  competitors/
  notes/
```

The exact structure is flexible; clarity and completeness matter more than folder names.
